What is Moore Miles?

Moore Miles is Moore Elementary’s signature annual fundraiser — a week-long celebration of community, fitness, and school spirit that ends with a Fun Run on campus.

Fun Run

Every student participates in a 1-mile run on campus.

School Spirit

Five themed spirit days build excitement and strengthen community.

$50k Goal

Funds classroom enrichment, field trips, and school supplies.

Auction

Family bake sale and a school-wide auction add to the fun.
